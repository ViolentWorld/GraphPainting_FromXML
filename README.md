## About:books:

Using this application, you can get a graphical representation of the **XML** file in the form of a tree, where the vertices are tags.

## Examples:eyes:

For example, the xml file contains a number structure:
```xml=
<Number>
  <Dozens>
    <Units> </Units>
  </Dozens>
  <Units>
    <Hundreds>
      <Dozens>
        <Units> </Units>
      </Dozens>
      <Eleven-Nineteen> 
      </Eleven-Nineteen>
      <Units>
      </Units>
    </Hundreds>
  </Units>
  <Eleven-Nineteen>
  </Eleven-Nineteen>
</Number>
```
